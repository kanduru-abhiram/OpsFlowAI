# OpsFlow AI Frontend

React 19 + TypeScript + Vite control tower for OpsFlow AI.

## Run

```bash
npm install
npm run dev
```

Set `VITE_API_BASE_URL` in `.env` when the backend is not running on `http://localhost:8000`.

## Build

```bash
npm run build
```
